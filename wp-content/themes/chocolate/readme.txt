“Chocolate WP” is compatible with the WordPress version 3.1 and later.

Most JavaScript enabled Browsers will make this theme work with no issues!

It has been tested on:
- Firefox 3.6 and later;
- Safari 4 and later;
- Opera 11.10 and later;
- Internet Explorer 8 and later;
- Google Chrome.

Before installing Chocolate WP, please make sure that following plugins are not installed or deactivated:
- tax-images;
- woo tumblog;
- wp-pagenavi.

After uploading the theme to your hosting you may need to:
1) The folder “your_site/wp-content/uploads” must be created. Make sure that CHMOD is set to 777.
2) Create folder and set its CHMOD to: “your_site/wp-content/uploads/dt_cache”.
3) Create folder and set its CHMOD to: “your_site/wp-content/uploads/dt_uploads”.
4) Set CHMOD 644 to the file “your_site/wp-content/themes/dt-chocolate/thumb.php”.
